Automation Management microservice

The Automation Management microservice.

The Automation Management microservice is responsible for all automation scenarios inside the Operating Block. It monitors and acts on the environment in such a way as to have the best environmental conditions available on the basis of needs. Moreover it handles the power-safe mode in the Operating Block in order to save precious energy maintaing at the same time keeping the environment in the right conditions. Finally it provides a support for the usage of medical technologies in order to adapt the environment when used.

To see the code documentation, check out the Javadoc page.
